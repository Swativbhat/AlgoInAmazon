Order Scheduling

1.Heap

A min-heap can efficiently manage order scheduling by always placing the earliest shipping date at the top, ensuring timely and priority-based dispatch of orders.

Code:


#include <iostream>
#include <vector>
#include <string>

using namespace std;

// Structure for a product
struct Product {
    string name;
    double price;
};

// Function to insert a new product into the min-heap
void insert_min_heap(vector<Product>& heap, Product newProduct) {
    // Add the new product to the end of the heap
    heap.push_back(newProduct);
    int index = heap.size() - 1;

    // Bubble up to maintain heap property (min-heap on price)
    while (index > 0 && heap[(index - 1) / 2].price > heap[index].price) {
        swap(heap[index], heap[(index - 1) / 2]);
        index = (index - 1) / 2;
    }
}

// Function to print the heap
void printHeap(const vector<Product>& heap) {
    for (const auto& product : heap) {
        cout << product.name << " ($" << product.price << ") ";
    }
    cout << endl;
}

int main() {
    vector<Product> heap;
    vector<Product> products = {
        { "Product A", 20.0 },
        { "Product B", 15.0 },
        { "Product C", 25.0 },
        { "Product D", 10.0 },
        { "Product E", 12.0 }
    };

    for (const auto& product : products) {
        insert_min_heap(heap, product);
        cout << "Inserted " << product.name << " ($" << product.price << ") into the heap: ";
        printHeap(heap);
    }

    return 0;
}

Time Complexity

Operation Average Worst Case
Insert O(log n) O(log n)
Delete O(log n) O(log n)

Space Complexity

Space O(n)

Key Inferences

  • Heaps prioritize orders by earliest shipping date for quick scheduling
  • They help ensure orders are dispatched on time and in the right order.
